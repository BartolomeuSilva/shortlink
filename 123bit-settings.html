Documentação da API

A API REST do 123bit.app permite criar, editar, listar e deletar links, consultar analytics e gerenciar domínios programaticamente.

Zona de perigo
Ações irreversíveis. Proceda com cuidado.
Exportar todos os dados
Baixe um arquivo ZIP com todos os seus links, cliques e analytics em formato JSON.
Desativar todos os links
Todos os seus links deixarão de redirecionar imediatamente. Pode ser revertido.
Cancelar assinatura
Você continuará com acesso Pro até o fim do período pago. Após isso, downgrade automático para Free.
Excluir conta permanentemente
Remove sua conta, todos os links, analytics e dados associados. Esta ação não pode ser desfeita.
